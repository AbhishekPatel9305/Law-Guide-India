# Law Guide India

Law Guide India is a web-based project designed to spread legal awareness and provide simplified information about Indian laws. The platform helps users understand their rights, duties, and basic legal procedures in an easy-to-understand format.

## Features
- Simple and user-friendly interface
- Information about Indian laws and rights
- Easy navigation and clean layout
- Responsive design for mobile and desktop

## Technologies Used
- HTML5
- CSS3
- JavaScript

## How to Run the Project
1. Download or clone the repository
2. Extract the project files (if downloaded as ZIP)
3. Open `index.html` in any web browser

## Project Goal
The goal of this project is to improve legal awareness among citizens and help them understand basic legal concepts without complexity.
